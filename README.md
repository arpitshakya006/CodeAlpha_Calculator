This is a simple and responsive Calculator built using HTML, CSS, and JavaScript. It performs basic arithmetic operations and provides a user-friendly interface with keyboard support.

Features
Addition (+)
Subtraction (-)
Multiplication (×)
Division (÷)
Clear Screen (C)
Backspace (⌫)
Keyboard Support
Responsive Design
Error Handling

Technologies Used
HTML5
CSS3
JavaScript (ES6)
